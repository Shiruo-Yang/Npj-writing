# Npj-writing

`Npj-writing` is a Codex skill for drafting, revising, and auditing manuscripts in an `npj Computational Materials` style, with a focus on materials informatics, machine-learning screening, applicability-domain control, and mechanism-aware two-photon photoinitiator discovery.

The skill is designed to keep three evidence roles separate:

- `npjCM writing corpus`: narrative structure, Results and Methods organization, reliability framing, and restrained contribution language.
- `2PI primary evidence`: two-photon polymerization, two-photon absorption, radical photoinitiator mechanisms, and optical-to-chemical initiation coupling.
- `ML QM discovery methods`: graph neural networks, scaffold-disjoint validation, ZINC22 screening, uncertainty, applicability domain, reproducibility, and QM triage.

## What it helps with

- Title and abstract drafting within NPJ Article constraints.
- Introduction restructuring without generic AI hype.
- Results organization by evidence modules rather than script chronology.
- Discussion claim calibration without unsupported experimental overclaiming.
- Methods checklists for reproducible ML, screening, and quantum-mechanical workflows.
- Figure narrative planning for computational materials discovery manuscripts.

## Current manuscript logic

The working 2PI manuscript should be framed as:

> Mechanism-aware and evidence-governed computational prioritization of radical two-photon photoinitiator candidates.

It should not be framed as experimentally validated photoinitiator discovery unless synthesis and photopolymerization evidence are included.

## Public note

This public repository contains the reusable writing framework only. Local corpora, manuscript drafts, extracted source texts, private notes, and project data are intentionally not included.
